## Exemplo de Sessao

```
1. websearch("awesome-mcp-servers site:github.com") -> encontra repos

4. Adicionar ao opencode.json com tags e env vars
5. npx test -> timeout esperado (MCP stdio server)
6. ConvertFrom-Json validate -> JSON OK
7. Backup -> opencode.json.backup
```
