## Pré-requisitos

|------------|--------------|------------|
| pandoc | 3.1+ | `winget install pandoc` ou manual |

| python-docx | 1.1+ | `pip install python-docx` |
| Python | 3.11+ | `pip install python-docx pandas numpy matplotlib seaborn plotly scikit-learn` |
