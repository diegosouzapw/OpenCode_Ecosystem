# Padrões de Excelência Qualis A1 e Auditoria Científica

Esta referência define os critérios rigorosos para a produção de conteúdo científico de alto impacto, servindo como guia para a auditoria e validação de resultados.

## 1. Rigor Metodológico e Estatístico
Para atingir o nível Qualis A1, a pesquisa deve demonstrar:
- **Validação Cruzada**: Uso de k-fold cross-validation ou conjuntos de teste independentes para garantir a generalização.
- **Análise de Poder**: Cálculo do poder estatístico (1-β) para justificar o tamanho da amostra e a confiabilidade dos resultados.
- **Tamanho do Efeito**: Reportar Cohen's d, Pearson's r ou outras métricas de magnitude, além do p-valor.
- **Correção para Múltiplas Comparações**: Aplicação de Bonferroni, Holm-Bonferroni ou FDR (False Discovery Rate) quando aplicável.

## 2. Estrutura e Didática PhD
O texto deve seguir a estrutura IMRAD (Introdução, Métodos, Resultados e Discussão) com as seguintes nuances:
- **Texto Corrido**: Evitar listas excessivas; preferir parágrafos densos e argumentativos.
- **Diálogo entre Autores**: A revisão de literatura deve ser um debate, não apenas um resumo de fontes.
- **Análise de Sensibilidade**: Testar como as conclusões mudam sob diferentes pressupostos ou parâmetros.
- **Limitações e Futuro**: Discussão honesta das fraquezas do estudo e proposição de experimentos futuros específicos.

## 3. Checklist de Auditoria (Constraints)
| Critério | Requisito Qualis A1 |
| :--- | :--- |
| **Originalidade** | Deve preencher uma lacuna teórica ou prática explicitamente identificada. |
| **Rastreabilidade** | Todas as afirmações devem ter citações verificáveis (DOI/URL). |
| **Reprodutibilidade** | Código e dados devem estar disponíveis ou descritos minuciosamente. |
| **Rigor Matemático** | Fórmulas devem estar em LaTeX e seguidas de explicação detalhada. |
| **Visualização** | Gráficos e tabelas devem ser autoexplicativos e de alta qualidade. |

## 4. Referências e Citações
- **Formato**: ABNT (Autor, Ano, p. X) ou conforme norma específica do periódico.
- **Quantidade**: Mínimo de 40-60 referências para artigos de nível doutorado.
- **Atualidade**: Pelo menos 50% das referências devem ser dos últimos 5 anos.
