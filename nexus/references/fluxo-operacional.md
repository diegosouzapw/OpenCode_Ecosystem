## Fluxo de Trabalho Operacional (Nexus-v6.1 - Claude Code Enhanced)

Para utilizar o NMA em uma tarefa complexa, siga rigorosamente este fluxo, que integra as capacidades MICRO-granulares do TMA:

1.  **Meta-Orquestração (L0):** Inicie o processo com `scripts/meta_orchestrator.py` para alinhar objetivos, decompor a tarefa em operações atômicas e estabelecer o gerenciamento de sessão e permissões, inspirando-se no `Session Manager` e `Permission Gate` do Claude Code. **Esta camada agora inclui um módulo de Auditoria Científica Qualis A1, que atua como um 'Harness de Veracidade' para validar a autenticidade e relevância de citações em tempo real, especialmente para o `criador-de-artigo-v2`.**
2.  **Descoberta e Raciocínio (L1-L2):** Utilize `scripts/domain_discovery_engine.py` para extração de conceitos, `scripts/micro_reasoning_types.py` para selecionar o tipo de raciocínio ideal (veja `references/reasoning.md` para detalhes dos 38 sub-tipos), e integre mecanismos de `Skill Registry` e `Context Compressor` para otimizar o conhecimento disponível.
3.  **Execução Sincronizada (L3-L5):** Implemente as barreiras de sincronização com `scripts/micro_sync_barriers.py` e valide cada passo usando `scripts/micro_validation.py` (consulte `references/barriers.md` para 120+ barreiras e 500+ restrições). A orquestração multiagente é aprimorada com conceitos de `Subagent Spawner`, `Teammate Mailboxes`, `FSM Protocol`, `Autonomous Board` e `Worktree Isolator` para delegação e coordenação eficientes.
4.  **Feedback e Evolução (L6):** Monitore a execução com `scripts/micro_feedback_loop.py` (120 feedback points) e gere otimizações contínuas usando `scripts/meta_learning_engine.py`. **Esta camada agora incorpora um Protocolo de Simulação de Banca Examinadora (Peer-Review) para o `criador-de-artigo-v2`, avaliando o artigo contra critérios Qualis A1 e fornecendo feedback iterativo para garantir a nota 10/10.** A `Observability Layer` (Event Bus, Background Executor) é integrada para um monitoramento abrangente e execução de tarefas em segundo plano.
